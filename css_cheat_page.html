Property Description Example
color Specifies the text color color: red;
font-family Sets the font family for text font-family: Arial, sans-serif;
font-size Sets the font size font-size: 16px;
margin Sets the margin of an element margin: 10px;
padding Sets the padding of an element padding: 10px;
background-color Sets the background color of an element background-color: #f1f1f1;
border Sets the border of an element border: 1px solid #ccc;
display Sets how an element is displayed display: block;
position Sets the positioning method of an element position: relative;
float Sets the floating behavior of an element float: left;
text-align Sets the text alignment of an element text-align: center;
opacity Sets the transparency level of an element opacity: 0.5;
transition Adds transition effects to elements transition: width 2s;
flexbox Layout mode for arranging items in a container display: flex;
grid Layout mode for arranging items in a grid display: grid;
animation Applies animations to elements animation: slide 1s ease-in-out;
